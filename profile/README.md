# Community Access

**Making software accessible by default, not as an afterthought.**

Community Access is a community-driven open-source organization building AI-powered tools that enforce accessibility standards in developer workflows. Our agents work inside Claude Code, GitHub Copilot, and Claude Desktop so that every line of code ships inclusive.

## What We Build

- **[Accessibility Agents](https://github.com/Community-Access/accessibility-agents)**: 35 specialized AI agents that enforce WCAG 2.2 AA compliance across web code, Office documents, PDFs, and Markdown. Install in one command.

## Why This Exists

AI coding tools generate inaccessible code by default. They forget ARIA rules, skip keyboard navigation, ignore contrast ratios, and produce modals that trap screen reader users. We built the agents we wished existed and opened them to the world.
